# Knowledge Index

Este archivo es el mapa de conocimiento del equipo. Cada agente lo lee para decidir que archivos cargar segun la tarea.

## Regla de carga
- INDEX.md siempre se carga (es ligero).
- Cada agente carga solo 2-4 archivos adicionales segun la tarea. NO leer todo el directorio.
