# Tu Equipo de Lanzamiento (version simple)

Imagina que contratas un equipo de 5 expertos en marketing.
Les das un brief, ellos trabajan en cadena, y tu apruebas cada paso.

## Que lo hace diferente de "pedirle a la IA que me escriba copy"

```
┌──────────────────────────────┐    ┌──────────────────────────────┐
│  IA generica                 │    │  Este sistema                │
│                              │    │                              │
│  - Empieza de cero           │    │  - Lee TU voz de marca       │
│  - No sabe tu tono           │    │  - Usa frameworks probados   │
│  - Cada pieza es aislada     │    │  - Todo esta conectado       │
│  - No aprende                │    │  - Aprende de cada lanzam.   │
│  - Output generico           │    │  - Output personalizado      │
│                              │    │                              │
│  "Escribeme un email         │    │  "El email 5 destruye la     │
│   de ventas"                 │    │   objecion 'no tengo tiempo' │
│                              │    │   usando el tono de marca    │
│                              │    │   y la estrategia aprobada"  │
└──────────────────────────────┘    └──────────────────────────────┘
```

---

## Los 5 comandos que necesitas

```
/nuevo-lanzamiento      Empezar un lanzamiento de cero
                        (te guia paso a paso)

/iterar                 Cambiar algo especifico
                        ("reescribe este email", "dame mas headlines")

/revisar-estrategia     Ajustar la estrategia de un lanzamiento

/exportar-lanzamiento   Ver todo junto en un documento

/cerrar-lanzamiento     Termino el lanzamiento, meter numeros reales
                        (el sistema aprende para el proximo)
```

---

## El secreto: mejora con cada lanzamiento

```
    Lanzamiento 1
    "Vendimos $5,000. Los emails de historia
     funcionaron mejor que los de objeciones."
                │
                ▼  (el sistema guarda esto)
                │
    Lanzamiento 2
    "El sistema ya sabe que los emails de historia
     funcionan mejor. Los prioriza. Vendimos $12,000."
                │
                ▼  (guarda esto tambien)
                │
    Lanzamiento 3
    "Ahora sabe que funciona y que no para MI marca.
     Vendimos $28,000."
                │
                ▼
          ...cada vez mas inteligente
```

Ningun freelancer ni agencia tiene esto: un historial de TUS datos
reales que mejora cada decision automaticamente.
